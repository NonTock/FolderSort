# Smart File Sorter 📂

Интерактивная консольная утилита на Python для автоматической сортировки файлов в указанной директории. Скрипт анализирует расширения файлов и бережно раскладывает их по соответствующим категориям (Папкам), помогая мгновенно навести порядок в "Загрузках" или на Рабочем столе.

## Особенности (Features)
* **Интеллектуальная сортировка:** Распознает и сортирует файлы по категориям: `Images`, `Documents`, `Videos`, `Archives`, `Executables`, `Torrents`, `Programming`. Неизвестные форматы отправляются в папку `Others`.
* **Защита от перезаписи:** Если в целевой папке уже существует файл с таким же именем, скрипт не удалит его, а приостановит работу и предложит ввести новое имя.
* **Визуализация процессов:** Красивый цветной интерфейс в консоли и динамический прогресс-бар обработки файлов.
* **Безопасность:** Программа игнорирует вложенные папки (сортирует только файлы верхнего уровня), что исключает риск поломки структуры директорий.

## Стек технологий
* **Python 3.x**
* `pathlib` — встроенная библиотека для кроссплатформенной работы с путями.
* `colorama` — для цветного вывода системных сообщений.
* `tqdm` — для отображения прогресс-бара во время сортировки.

## Установка и запуск

1. Склонируйте репозиторий на свой компьютер:
   `git clone https://github.com/NonTock/FolderSort.git`

2. Перейдите в папку с проектом:
   `cd FolderSort`

3. Установите необходимые зависимости из файла requirements.txt:
   `pip install -r requirements.txt`

4. Запустите скрипт:
   `python main.py`

## Как пользоваться
1. При запуске скрипт спросит, хотите ли вы начать работу. Введите `1`.
2. Вставьте полный путь к папке, которую нужно отсортировать (например, `C:/Users/Username/Downloads`).
3. Скрипт проанализирует папку и выведет список файлов с их будущими категориями.
4. Подтвердите начало сортировки и дождитесь заполнения прогресс-бара. Готово!
